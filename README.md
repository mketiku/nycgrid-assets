# nycgrid-assets

Static assets for [NycGrid](https://github.com/mketiku/interesting), served via jsDelivr CDN.

Pin to a tag — tagged URLs are immutable:

```
https://cdn.jsdelivr.net/gh/mketiku/nycgrid-assets@v1.1.0/path/to/file
```

## Contents

```
audio/ambient/   lofi_jazz, lofi_terminal, lofi_crosswalk, lofi_hiphop, lofi_elevator (.mp3)
audio/podcast/   fresh-asphalt-ep1, stoop-talk-ep1 (.m4a)
```
